# FLYP-EC2-Repo
Webapp for FLYP capstone project

### Setup

```
npm install
pm2 start node_modules/react-scripts/scripts/start.js --name Webapp
```

#### Control
```
pm2 list // Shows status
pm2 start API
pm2 restart API
pm2 stop API
```

### Notes
* Defaults to port 3000
